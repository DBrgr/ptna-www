Are there other tools on OSM which support Quality Assurance for Public Transport?

Sure there are ...

  • pt_assistant Plugin of JOSM
  • OSM Inspector provides an Analysis of Public Transport Routes and an Analysis of Public Transport Stops
  • Examples belonging to Overpass API
  • OSM Relation Analyzer and Documentation in the OSM Wiki
  • French colleagues provide a tool for the Analysis of Route-Relations
  • French colleagues provide another tool for the Analysis of Route-Relations
  • Ilya Zveriks Subway Validator
  • Quality Assurance in the OSM Wiki
