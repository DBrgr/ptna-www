Resultados para o Brasil

A primeira coluna inclui um link para os resultados da análise.

A coluna "Últimas alterações" está vinculada a uma página HTML que mostra as diferenças para os últimos resultados da análise. São coloridas, você pode usar os botões de navegação na parte inferior direita para pular de uma diferença para outra. Essas colunas incluem a data da última análise em que surgiram alterações relevantes. Datas anteriores significam que não houve alterações nos resultados. No entanto, os dados foram analisados conforme indicado na coluna "Data da análise".
